# Project Memory

## Core
RTL Arabic blood donation app for Al-Qarara. Tech: React 18, TS, Vite, Tailwind, shadcn, Supabase, Recharts.
Donor eligibility strictly >= 90 days since last donation.
Blood type colors: O (Blue), A (Green), B (Red), AB (Purple).
Design: Glassmorphism, red liquid wave bg, frosted cards, pill inputs, deep red buttons.
